# pp v1.0 Acceptance Checklist

`pp` v1.0 is an open-source-ready static release of the prompt launcher.

## Included

- Dependency-free browser app under `prototype/`.
- Default GitHub repo set to `https://github.com/PoplarYang/prompts`.
- GitHub sync settings and offline browser cache.
- jsDelivr fallback for public GitHub sync when GitHub tree API is unavailable or rate-limited.
- Demo prompt repository content under `demo-repo-content/`.
- Reusable prompt repository template under `prompt-repo-template/`.
- README, contribution guide, code of conduct, changelog, release guide, and MIT license.
- Parser and ranking tests.
- Versioned verification gate through `python3 scripts/verify_all.py`.
- Release package script through `python3 scripts/package_release.py`.
- GitHub Actions verify and release workflows.
- Screenshot assets under `docs/assets/`.

## Deferred To Native Desktop Track

- Native global shortcut.
- Native macOS and Windows installers.
- Native tray/menu behavior.
- Tauri shell and OS clipboard APIs.

The static release is intentionally useful now while keeping the native desktop path clear.

## Verification

```sh
python3 scripts/verify_all.py
python3 scripts/package_release.py
```
