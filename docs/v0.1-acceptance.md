# pp v0.1 Acceptance Checklist

`pp` v0.1 is a dependency-free local browser app that proves the core prompt launcher experience before the Tauri desktop shell is introduced.

## Included

- Reads prompt content from Markdown files with YAML frontmatter.
- Builds a local JSON prompt index.
- Loads the generated index in the browser app.
- Searches title, description, tags, aliases, category, path, and body.
- Renders Markdown preview with code blocks.
- Copies the selected prompt body to the clipboard.
- Shows a selected-text manual copy fallback if browser clipboard access is blocked.
- Persists favorites, usage count, and last-used time in local browser storage.
- Supports keyboard navigation.
- Works offline after the local files are present.
- Provides a local verification command.

## Deferred

- Native desktop shell.
- Global system shortcut.
- GitHub sync.
- Custom repository settings.
- Native OS clipboard integration outside the browser.
- App packaging for macOS and Windows.

These are still part of the broader product direction, but v0.1 intentionally validates the launcher loop first.

## Commands

Build the prompt index:

```sh
python3 scripts/build_prompt_index.py
```

Verify v0.1 artifacts:

```sh
python3 scripts/verify_v0_1.py
```

Run the local app:

```sh
python3 -m http.server 8765
```

Open:

```txt
http://localhost:8765/prototype/
```
