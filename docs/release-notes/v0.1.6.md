# pp v0.1.6

## 中文

### macOS 发布方式

- macOS 发布包以 ZIP 为主，分别提供 Apple Silicon (`aarch64`) 和 Intel (`x64`) 版本。
- 当前版本未使用 Apple Developer 签名和 notarization，首次打开时需要手动允许。
- 安装步骤：下载对应架构的 ZIP，解压后将 `pp.app` 拖到“应用程序”，右键点击应用并选择“打开”，再确认打开。
- 如果系统仍显示“应用已损坏”，在终端执行：

```bash
xattr -cr /Applications/pp.app
open /Applications/pp.app
```

### 其他改进

- GitHub Actions 增加 macOS ZIP/DMG、Windows 安装包和单元测试校验。
- macOS Release 同时构建 Apple Silicon 和 Intel 版本。

## English

### macOS distribution

- ZIP is the primary macOS distribution format, with separate Apple Silicon (`aarch64`) and Intel (`x64`) builds.
- This release is not signed with Apple Developer ID and is not notarized, so the first launch requires manual Gatekeeper approval.
- Installation: download the ZIP for your Mac, extract it, move `pp.app` to Applications, then right-click the app, choose Open, and confirm.
- If macOS still reports that the app is damaged, run:

```bash
xattr -cr /Applications/pp.app
open /Applications/pp.app
```

### Other improvements

- GitHub Actions now verifies macOS ZIP/DMG packages, Windows installers, and release unit tests.
- macOS releases are built for both Apple Silicon and Intel.
